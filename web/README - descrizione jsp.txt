Descrizione Contenuto Java Server Pages

Bacheca -> Visualizzazione di una Bacheca, sia essa personale o di un gruppo

deleteGroupButton -> Form, con parametri hidden e un solo bottone submit, per l'eliminazione di un gruppo

descrizione -> Start page di Nerdbook

footer -> descrizione del footer

headerBacheca ->
headerDescrizione ->
headerLogin ->
headerProfilo ->
headerRicerca -> descrizioni degli header delle varie pagine, 
		 con eventuali dettagli sull'utente loggato e barra di navigazione

listaPost -> tabella contenente post di una data bacheca

login -> pagina per il login di un utente già iscritto

registrazione -> pagina per la registrazione di un nuovo utente

ricerca -> pagina con input collegato a javascript per la ricerca di utenti e gruppi

wallActions -> controllo e decisione sulle azioni da offrire all'utente 
	       durante la visita di una bacheca (richiesta/rimozione di una amicizia/iscrizione)

wallOwnerInfos -> presentazione dei dettagli sul proprietario della bacheca visitata

wallListJson -> presentazione di una lista di oggetti json
